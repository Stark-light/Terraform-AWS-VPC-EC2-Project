# Terraform-AWS-VPC-EC2-Project
Terraform project that provisions an AWS VPC, subnet, security group, and EC2 instance using Infrastructure-as-Code.

## Resources Created
- VPC
- Public Subnet
- Internet Gateway
- Route Table
- Security Group (SSH access)
- EC2 Instance (Free Tier eligible)

## Prerequisites
- AWS account
- AWS CLI configured
- Terraform installed
- EC2 Key Pair created in AWS

## Steps to Run
1. terraform init
2. terraform plan
3. terraform apply

## Purpose
This project was created for learning Infrastructure-as-Code and basic cloud networking concepts.
